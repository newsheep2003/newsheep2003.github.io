    Robotmaster视觉组环境安装教程

    发表于 2023-10-17 更新于 2025-03-28 分类于 笔记

    写一份笔记以备部署到小电脑，或者是重装系统，顺便给学弟当教程。

    系统安装

    Linux Ubuntu 20.04

    软件安装

    VScode

    01.依赖软件安装
    sudo apt install software-properties-common apt-transport-https wget

    02.使用 wget 命令插入 Microsoft GPG key ：
    wget -q https://packages.microsoft.com/keys/microsoft.asc -O- | sudo apt-key add -

    03.启用 Visual Studio Code 源仓库，输入：
    sudo add-apt-repository "deb [arch=amd64] https://packages.microsoft.com/repos/vscode stable main"

    04.一旦 apt 软件源被启用，安装 Visual Studio Code 软件包：
    sudo apt install code

    Git

    01.软件安装

    sudo apt install git

    02.用github上注册的用户名和邮箱地址，配置git

    git config --global user.name "username"
    git config --global user.email "username@email.com"

    03.重启ubuntu，并查看git是否配置成功

    sudo git config --global --list

    如果没有重启，查看git是否配置成功可能报错：

    fatal: 无法读取配置文件 '/root/.gitconfig': 没有那个文件或目录

    04.生成密钥

    生成 ssh 密钥，ed25519 也可以用 rsa，只是代表两种不同算法生成的密钥

    ssh-keygen -t ed25519 -C "username@email.com"

    response 如下，设置私钥的文件名（产生一个私钥，和一个同名的.pub结尾公钥）和通行短语。不设置可以回车跳过

    使用cat命令查看生成的pub密钥

    cat /home/username/.ssh/id_ed25519.pub

    此处为Ubuntu用户名，不是GitHub用户名

    OpenVINO

    解决 error: invalid command 'bdist_wheel' 的方法

    先临时换源升级pip python -m pip install -i https://pypi.tuna.tsinghua.edu.cn/simple --upgrade pip

    再永久换源 pip config set global.index-url https://pypi.tuna. tsinghua.edu.cn/simple
